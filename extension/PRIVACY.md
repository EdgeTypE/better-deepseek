# Privacy Policy for Better DeepSeek

**Last updated:** 5 May 2026

Better DeepSeek ("the Extension") respects your privacy and is fully committed to protecting your personal data. 

**1. Data Collection & Storage**
The Extension does not collect, transmit, distribute, or sell any of your personal data, chat logs, or browsing history to any external servers or third parties. All data generated by the Extension, including custom characters, memory items, skills, local workspace metadata, and optional settings such as a GitHub personal access token, is stored strictly locally on your own device using browser storage APIs such as `chrome.storage.local`.

If you choose to save a GitHub personal access token for private repository fetching, that token is stored locally by the extension and is not encrypted at rest by the extension itself. It is never shown in the UI unless you explicitly reveal it, and it is not included in the extension's import/export features.

**2. Third-Party Services**
The Extension operates on `chat.deepseek.com`. Please refer to DeepSeek's own privacy policy regarding how they handle your chat data. To provide advanced features (like Python execution or HTML rendering), the Extension may fetch open-source libraries from standard CDNs (e.g., PyScript, jsDelivr).

If you explicitly use the GitHub repository import feature, the Extension may contact GitHub over HTTPS to download repository archives. When you have configured a GitHub token, the token is used only for authenticated GitHub repository requests and is sent only to GitHub's repository download endpoint.

**3. Permissions**
- `storage`: Used solely to save your local extension settings.
- `host_permissions`: Used to interface with the DeepSeek chat DOM, to allow the "Web Fetch" tool to read sites you explicitly ask it to summarize, and to download GitHub repository archives you explicitly request.

If you have any questions or concerns, please open an issue on our GitHub repository.
